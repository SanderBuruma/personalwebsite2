The idea of the minimalistic website

This website is intended to be minimalistic. A minimalistic website is meant to stay very small and very simple, similar to how Luke Smith describes it on YouTube.

A simple website is a website which is easily loaded on any device anywhere because it is small. In addition any device can display it because it only uses html/css and nothing fancy. As it stands today many websites cannot function efficiently for people in developing nations without access to fast internet plus the latest smartphones and laptops. This should not be the case. The majority of the internet should be easily usable by more than 95% of the world's population; hence minimalism.
